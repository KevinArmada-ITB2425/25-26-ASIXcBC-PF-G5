# Justificació Tècnica — Competències ASIX

| Mòdul ASIX | Component del Projecte |
|---|---|
| **Sistemes Operatius en Xarxa** | Instal·lació i hardening de Debian/Ubuntu, gestió de serveis systemd, SSH amb clau pública |
| **Planificació i Administració de Xarxes** | Disseny de topologia, subneting, VLANs, routing estàtic, NAT a pfSense |
| **Seguretat i Alta Disponibilitat** | Firewall perimetral, IDS/IPS Suricata, SOC amb Wazuh, VPN WireGuard, mínim privilegi |
| **Serveis en Xarxa** | Kea DHCP per VLAN, Unbound DNS amb DNSSEC i filtratge, integració de serveis |
| **Implantació d'Aplicacions Web** | Accés a serveis interns via VPN, gestió remota de firewall i SOC via WebGUI |

## Competències Transversals

- **Gestió d'Incidents**: detecció, anàlisi i resposta mitjançant Wazuh (mapeig MITRE ATT&CK)
- **Documentació tècnica**: repositori GitHub amb Conventional Commits, diagrames i memòria
- **Autonomia i resolució de problemes**: disseny i implementació end-to-end sense infraestructura comercial
